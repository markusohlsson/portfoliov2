## ğŸš€ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/markusohlsson/portfoliov2.git
cd portfoliov2
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

Run linters:

